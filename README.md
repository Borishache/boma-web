# BOMA - Agencia de Marketing

Este es el sitio web oficial de BOMA, una agencia de marketing enfocada en emprendedores y pequeños negocios.

## Estructura del Proyecto

- `index.html`: Página principal.
- `style.css`: Estilos y diseño visual.
- `script.js`: Funcionalidades interactivas.
- `logo.svg`, `mark.svg`: Assets de marca.

## Cómo Publicar en GitHub Pages

1. Crea un nuevo repositorio en [GitHub.com](https://github.com/new).
2. Sube todos los archivos de esta carpeta al repositorio.
3. Ve a **Settings (Configuración)** > **Pages**.
4. En **Source**, selecciona `main` branch y guarda.
5. Tu sitio estará vivo en unos minutos en el enlace que te proporcionará GitHub.
